HTML

HTML singkatan dari HyperText Markup Language, HTML ini berguna untuk struktur awal dari sebuah website seperti tulisan, input, tombol, icon, gambar, tabel dsb

Tag Utama HTML :

  1. <!DOCTYPE html> yang mana ini digunakan untuk memberitahu browser bahwa yang digunakan adalah HTML5.

  2. <head> head bisa kita gunakan untuk SEO atau search engine optimization yang berguna agar web kita mudah di temukan oleh orang lain di browser, selain itu head juga bisa kita gunakan untuk menautkan dengan css, js, ataupun dengan link icon, cdn, maupun font.

  3. <tittle> title berada di dalam <head> yang berfungsi untuk memberikan tab tulisan yang biasanya ada di bagian atas browser. Seperti web ini, web ini memiliki tittle 'Mini Task CCI'

  4. <body> nahh untuk body, body ini adalah inti dari web kita dimana seluruh isi web kita di tuliskan dalam tag body seperti kita menambahkan foto, form, ataupun teks yang nantinya muncul di browser pengguna. Intinya tag ini berisi semua dari web kita kecuali judul di bagian atas browser karena itu bagian dari tittle.

CSS

CSS singkatan dari Cascading Style Sheets, sesuai namanya 'style' dimana css ini di gunakan untuk memberikan style pada html untuk mempercantik web kita, css juga memiliki framework seperti bootstrap,tailwind, dan bulma. Tetapi yang sering saya gunakan adalah tailwind, kenapa? karena tailwind memberikan kita kebebasan untuk mendesain web kita, jadi kita memiliki kebebasan penuh terhadap design yang kita buat, bahkan web ini saja saya buat menggunakan tailwind.

Cara Penggunaan :
  • Inline CSS inline css adalah penggunaan css di dalam satu baris tag html yang akan kita beri style berikut contohnya : <p style="color : red;">
  • Internal CSS internal css adalah penggunaan css di dalam satu folder dengan html, cara penulisannya adalah di bagian head kita bisa menambahkan tag <style></style> lalu kita bisa menambahkan properti dan value dari design yang kita buat. untuk memilih tag mana yang ingin kita beri style kita perlu memberikan selector pada tag htmlnya dengan menggunakan class atau id, tetapi biasanya saya menggunakan class karena class bisa kita akses berulang kali sedangkan id hanya bisa di akses 1 kali pada css, kita juga perlu memberikan class yang sesuai dengan konteks, setelah kita berikan nama class baru kita bisa panggil class tersebut di css dan bisa kita beri styling sesuai kebutuhan.
  • Eksternal CSS eksternal css hampir mirip dengan internal bedanya di eksternal css sintaks/kode css berada pada file yang terpisah. persamaannya terletak pada pemilihan tag yang akan kita design yaitu menggunakan selector, selain menggunakan class dan id, kita juga bisa menggunakan selector tag itu sendiri, misalnya jika kita ingin memberikan style pada tag <p> kita hanya perlu memanggil tag itu dengan p {lalu bagian ini di isi oleh properti dan value css}. Tetapi perlu diingat jika kita memiliki tag p yang lebih dari 1 pada html, maka semua tag p akan mendapat design yang sama
  • Contoh Kode :

    h1 {
    background-color:black;
    color:white;
    margin:100px;
    padding:30px;
    }

  • Penjelasan

    Bisa dilihat di atas saya menggunakan selector h1 yang berarti h1 di html yang akan kita beri style, di dalam blok kode terdapat background-color yang berarti bagian belakang dari teks h1, lalu diikuti dengan value black yaitu warna bagian belang h1 berwarna black atau hitam, lalu ada properti color properti ini digunakan untuk mengganti warna teks di dalam tag h1 menjadi berwarna putih, lalu margin dan padding untuk memberi jarak, margin untuk memberi jarak di luar elemen. sedangkan padding memberi jarak di dalam elemen.

  • let

    nah let ini jenis variabel yang sering digunakan, let bersifat blok scope, dan mirip dengan var dimana kita bisa mengganti nilai yang sudah di deklarasikan

  • const

    jenis const ini tuh singktannya konstanta artinya jika variabel di deklarasikan dengan const maka nilai dari variabel tersebut tidak bisa di rumah, jika kalian merubahnya maka akan terjadi eror.
